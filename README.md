# 💈 Glamour Touch Salon — AI Website Copy Generator

## About This Project
This project was built as part of a Prompt Engineering internship task.
The goal was to design structured prompts that generate high-quality 
website copy for a local business using AI tools like Claude and ChatGPT.

---

## 🏪 Business Chosen
- **Business Name:** Glamour Touch Salon
- **Location:** Madurai, Tamil Nadu
- **Type:** Men's Salon

---

## 🎯 Objective
Design repeatable prompts that produce:
- Homepage copy with strong value proposition
- Service descriptions
- Persuasive Call-to-Action (CTA) sections

---

## 🧠 Prompt Logic
Each prompt follows this structure:
- **Role** → Tell AI to act as a professional copywriter
- **Context** → Give business details
- **Task** → Specify exactly what to generate
- **Tone** → Define the feeling (confident, friendly, modern)
- **Format** → Specify how the output should look

---

- Claude AI → https://claude.ai
- ChatGPT → https://chat.openai.com
